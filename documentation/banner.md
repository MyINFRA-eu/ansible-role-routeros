# Banner Tasks

Set the login note (banner) show on routeros (mikrotik) devices

## Tags

- routeros_banner

## Settings

### Note (banner) to show on login

```yaml
routeros_banner:
  "Welcome to this device"
  "This device is managed by:"
  " - User 1"
  " - User 2"
```

```json
routeros_banner: [
  "Welcome to this device"
  "This device is managed by:"
  " - User 1"
  " - User 2"
]
```
