# Banner Tasks

Set the login note (banner) show on routeros (mikrotik) devices

## Tags

- routeros_banner

## Settings

### Note (banner) to show on login

```yaml
routeros_banner:
  "Welcome to this device \n"
  "This device is managed by: \n"
  " - User 1 \n"
  " - User 2 \n"
```

```json
"routeros_banner": [
  "Welcome to this device \n",
  "This device is managed by: \n",
  " - User 1 \n",
  " - User 2 \n"
],
```
